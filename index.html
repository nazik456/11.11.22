Git Instructions

How to create local repository

  1. Right click IT Folder using the mouse
  2. Select "Git Bash Here
  3. Execute "mkdirfolderName "
  4. Execute "cd FolderName
  5. Execute"git init"

How to link Local repo with Github

  1. Go to New repository page
  2. Execute"git remote add origin https://github.com/userName/FolderName.git"
  3. Execute"git branch -M main"
  4. Execute"git push -u origin main"
